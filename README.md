## 📃 시작 가이드


### Installation

```bash
# required node >= v18.x.x
```

```bash
git clone https://github.com/DaHoon06/devwiki-mollrang.git

cd ./mollrang

npm ci
```

### Mollrang Server Run

```bash
cd ./mollrang

npm run start
```
